#Notes
A simple music player

The notes get higher from 'A' to ';' (or any other row)

#Compiling
Dependencies

* SFML 2+

Build system: cmake 3.1+
On Linux/OS X:

```
cd Notes                        #Go the the project directory
mkdir build/ && cd build/
cmake .. && make
./notes                         #Launch
```
